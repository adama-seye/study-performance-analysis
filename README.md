## Study Performance Analysis

This project analyzes the UCI Student Performance dataset to identify which academic and socio-demographic factors most strongly influence students' final grades.

### Methods
- Data cleaning and exploration using Pandas
- Correlation analysis with Seaborn heatmaps
- Linear regression modeling with Scikit-Learn

### Key Insight
Early-term grades (G1, G2) were stronger predictors of final performance than study time alone.
